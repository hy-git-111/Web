body

  • ol tag : ordered list, 정돈된 리스트
  • ul tag : unordered list, 정돈되지 않은 리스트
  • li tag : list, ol/ul tag의 자식태그
  • br tag : line break, 닫는 태그기 없는 빈 태그
  • p tag : paragraph, 단락
  • h tag : heading, 제목, h1~h6
  • img tag : image, attribute와 함께 사용됨

      • img tag의 attribute

    • src : source, src=""
    • width : width=""
